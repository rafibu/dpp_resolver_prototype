S4: Query Evaluation

S4 evaluates platform-local derived query access in two categories: predicate queries (SELECT / COUNT / SUM with AND-connected filters) and reverse-traverse queries (find current source revisions referencing a target). Each query is run in both INDEXED and ON_DEMAND execution modes and the results are compared for indexed/on-demand equivalence, with timing and result export.
